# Ziqx AI Skills

A collection of AI skills for Ziqx packages, providing instructions and examples for AI agents to understand and use Ziqx libraries effectively.

## Installation

To add these skills to your AI agent, run the following command:

```bash
npx skills add ziqx/ziqx-ai-skills
```

## Available Skills

- **ziqx-auth**: Comprehensive instructions and examples for `@ziqx/auth-js` integration.
- **ziqx-drive**: Documentation and usage patterns for the `@ziqx/drive-js` SDK.
